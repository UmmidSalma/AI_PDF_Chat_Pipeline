# ai-pdf-chat-pipeline

An AI-powered pipeline to chat with your PDFs, right in the browser.

## Tech Stack
- HTML / CSS / JS
- Python 3.8+
- Jupyter Notebook
- AI / ML backend

## Project Structure
- `pdf_chat.html` — Frontend interface, open in any browser
- `pipeline_with_backend.ipynb` — AI pipeline + backend logic

## Quick Start

### HTML interface
```bash
python -m http.server 8000
# open http://localhost:8000/pdf_chat.html
```

### Notebook
```bash
pip install jupyter
jupyter notebook
```

## Notes
- Check notebook cells for API keys before sharing
- If a backend is needed, look for `app.py` or a startup cell
- `requirements.txt` can be generated on request
